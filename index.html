IoT: Arduino ESP8266 + Firebase Realtime Database

FabLab Moscu

May 17, 2019

Step 1: Creatinging Firebase Realtime Database

1. Create Google account (if you don't have it yet).

2. Go to console.firebase.google.com

3. Create new project (give it a name and chose location).

4. Go to Database

5. Create Realtime Database.

6. Change permissions.

Step 2: Arduino ESP8266

0. Download Arduino studio HERE

1. Copy link below:

http://arduino.esp8266.com/stable/package_esp8266com_index.json

2. Go to File -> Preferences (Файл -> Настройки).

3. Paste link into "Additional Boards Manager URLs":

4. Go to Tools/Boards/Boards Manager..(Инструменты/Плата:/Менеджер плат...):

5. Search for "esp" and click "Install":

6. Download & Install FirebaseArduino library HERE.

7. Download & Install ArduinoJSON library HERE.

5. Create new Arduino Sketch and paste code from bellow:

6. Go to Firebase Console -> Настройки проекта -> Сервисные аккаунты -> Секреты базы данных.

7. Copy Database Secret and paste it in line №6 of your Arduino Sketch:

8. Copy Database Name and paste it in line №5 of your Arduino Sketch:

9. Set name of your WiFi network in line №7 and password in line №8 of your Arduino Sketch.

10. Choose Board "Node MCU 1.0" and avaliable port (if port is not available, try different USB ports several times).

11. Upload code to the board via USB.

12. Go to Fireebase Console -> Database [Temperature is incrementing every 5000ms]:
